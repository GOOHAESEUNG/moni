## Design Context

### Users
- **Primary A — 학생 (초등~중등)**: 학원에서 선생님이 배정한 개념을 AI(무니)에게 설명하는 역할. 성취감과 재미가 학습 지속의 핵심. 집중이 흐트러지기 쉬우므로 UI가 복잡하면 안 됨.
- **Primary B — 선생님/학원 강사**: 학생별 이해도 리포트 확인, 단원 관리. 신뢰감과 효율이 중요. 대시보드는 정보가 많아도 읽기 쉬워야 함.
- **Context**: 학원 수업 후 자습, 또는 숙제로 사용. 모바일/태블릿 비율 높음.

### Brand Personality
**호기심 · 따뜻함 · 신뢰**

- 무니는 달에서 온 아기 토끼 — 모르는 게 당연하고, 배우고 싶어함
- 학생이 "가르치는 사람"이 되는 순간의 뿌듯함을 디자인으로 강화
- 귀엽지만 장난감이 아닌, 실제로 쓰이는 학습 도구

### Aesthetic Direction
- **레퍼런스**: Duolingo — 캐릭터 중심, 게임화, 밝고 에너제틱한 학습 앱
- **안티레퍼런스**: 보라 그라디언트 남발, Inter 폰트, 카드 안에 카드, 딱딱한 SaaS UI
- **라이트 모드 전용** — 초등학생 타겟, 무니 캐릭터(밝은 색)에 최적
- **대화 화면**: 우주/달밤 배경 — 별빛 파티클, 어두운 그라디언트 하늘. 무니가 달에서 온 설정과 일치
- **폰트**: Nunito — 둥글고 친근, Duolingo 느낌. 한국어 폴백은 Rounded Mplus 1c
- **색상**: 달 옐로우 primary (#E8C547), 쿨 화이트 배경 (#F2F2F5), 쿨 그레이 secondary (#9EA0B4)
- **Radius**: 매우 큰 radius (1rem base) — 버튼, 카드 모두 pill/rounded-3xl 수준
- **그림자**: 컬러 그림자 (`shadow-primary/10`) — 흰 박스 그림자 금지

### Design Principles

1. **무니가 주인공이다** — 모든 화면에서 무니 캐릭터가 중심. 텍스트보다 캐릭터가 먼저 눈에 들어와야 함.

2. **학생은 항상 이긴다** — 막혀도, 틀려도 격려. 에러 메시지조차 무니가 위로하는 톤. 빨간색 경고보다 따뜻한 오렌지/골드.

3. **집중을 방해하지 않는다** — 대화 화면에서 UI 크롬 최소화. 무니와 입력창만 보여야 함. 내비게이션 숨김.

4. **선생님은 한눈에 읽는다** — 대시보드는 스캔 가능한 구조. 숫자와 색상으로 이해도를 즉시 파악.

5. **애니메이션은 목적이 있다** — 무니의 표정 전환, 별빛 파티클, 메시지 등장만. 장식용 애니메이션 금지.
