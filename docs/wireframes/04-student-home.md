# 04. 학생 홈

## 사용자 & 목적
- **사용자**: 학생
- **목적**: 오늘 배울 단원 확인 + 무니와 대화 시작
- **감정 목표**: 설렘, 가볍게 시작할 수 있다는 느낌

## ASCII 와이어프레임

```
┌─────────────────────────┐
│  안녕, [이름]! 🌙        │  ← 환영 인사
│  오늘도 무니를 도와줄래요? │
│                          │
│  ┌───────────────────┐   │
│  │    [무니 hero]    │   │  ← 달 위의 무니 이미지
│  │    w:200 h:200    │   │     float 애니메이션
│  └───────────────────┘   │
│                          │
│  ────오늘의 학습────       │
│                          │
│  ┌───────────────────┐   │
│  │ 🌙 분수의 덧셈     │   │  ← 오늘 단원 카드
│  │ 선생님: 홍길동 T   │   │
│  │ 예상 시간: 10분    │   │
│  │                   │   │
│  │ [무니에게 알려줘 →]│   │  ← CTA 버튼
│  └───────────────────┘   │
│                          │
│  ────지난 학습────         │
│                          │
│  ┌──────┐ ┌──────┐       │
│  │ 📚   │ │ 📚   │       │  ← 이전 세션 카드 (작게)
│  │ 소수  │ │ 분수  │       │
│  │ 92점 │ │ 75점 │       │
│  └──────┘ └──────┘       │
│                          │
│  [🏠] [📊] [👤]          │  ← 하단 네비게이션
└─────────────────────────┘
```

## Stitch 프롬프트

**앱 타입**: 모바일 학습 앱 홈 화면 (375px)
**무드**: 밝고 따뜻한 라벤더, 귀엽고 설레는 느낌

### 배경
크림 화이트 `#FAF9FF`에서 연라벤더 `#F0ECFF`로 이어지는 부드러운 그라디언트 배경.

### 상단 인사 영역
- "안녕, 지수! 🌙" — Nunito Bold 24px, 딥 라벤더 네이비
- "오늘도 무니를 도와줄래요?" — Nunito 16px, muted foreground

### 무니 히어로 이미지
- 달 위에 앉은 아기 토끼 이미지, 중앙 정렬, 200×200px
- 위아래로 부드럽게 float하는 애니메이션 (CSS transform translateY)
- 이미지 아래에 골드 별빛 파티클 2~3개

### 오늘의 학습 카드
- 섹션 타이틀: "오늘의 학습" — Nunito SemiBold 14px, muted
- 카드: 흰색 배경, radius 24px, `box-shadow: 0 8px 32px rgba(139,92,246,0.12)`, 패딩 20px
  - 상단: 🌙 이모지 + 단원명 "분수의 덧셈" (Nunito Bold 20px)
  - 중간: "선생님: 홍길동 T" + "예상 시간: 10분" (small, muted)
  - 하단: "무니에게 알려줘 →" 버튼 — 라벤더 퍼플 배경, 흰색 텍스트, radius 999px, 전체 너비

### 지난 학습 섹션
- 작은 카드 2개 가로 나열
- 각 카드: 과목 이모지 + 이름 + 점수 (라벤더 pill 형태)

### 하단 네비게이션
- 3개 아이콘 (홈, 통계, 프로필), 라벤더 배경, radius 20px
- 활성 탭: 라벤더 퍼플 아이콘, 비활성: 회색
