# 05. 대화 화면 (핵심 ⭐)

## 사용자 & 목적
- **사용자**: 학생
- **목적**: 무니에게 오늘 배운 개념을 설명하며 빈틈을 발견
- **감정 목표**: 집중, 뿌듯함, 가르치는 즐거움

## 컴포넌트 명세

| 컴포넌트 | 설명 |
|---------|------|
| 배경 | 딥 네이비 우주 배경 + 별빛 파티클 애니메이션 |
| 주제 pill | 상단 고정, 반투명 glassmorphism |
| 무니 캐릭터 | 140×140, 표정 전환 (AnimatePresence) + float 애니메이션 |
| AI 말풍선 | 왼쪽 정렬, 라벤더 반투명 배경, TTS 버튼 포함 |
| 학생 말풍선 | 오른쪽 정렬, 흰색 배경 |
| 입력창 | 하단 고정, 마이크 버튼 + 텍스트 입력 + 전송 |
| 이해도 바 | (선택) 상단에 얇은 progress bar로 무니 이해도 표시 |

## Stitch 프롬프트

**앱 타입**: 모바일 AI 학습 채팅 앱 (375×812px, iPhone 14 기준)
**무드**: 달밤 우주, 신비롭고 따뜻함, 집중감

### 배경
깊고 어두운 우주 하늘 배경. 상단에서 하단으로 `#0D0B1E → #1A1040 → #2D1B69` 그라디언트. 작고 빛나는 별들이 배경에 흩어져 있음 (white dots, 10% opacity varying). 화면 가장자리에 옅은 라벤더 글로우 효과.

### 상단 영역 (고정 헤더)
- 상단 중앙에 pill 형태의 주제 태그: `"🌙 분수의 덧셈"`, 배경 glassmorphism (white 15% opacity, blur 10px), radius 999px, 라벤더 보더, 높이 36px
- 오른쪽 상단에 작은 X 버튼 (대화 종료)

### 무니 캐릭터 영역
- 화면 상단 30% 위치에 아기 토끼 캐릭터 이미지 (140×140px), 중앙 정렬
- 캐릭터 아래에 작은 텍스트 라벨: `"모르겠어요... 🌙"`, Nunito 13px, 라벤더 색상

### 대화 메시지 영역
- 스크롤 가능한 메시지 목록, 하단에서 시작
- **AI 말풍선 (무니)**: 왼쪽 정렬, 배경 `rgba(139, 92, 246, 0.15)` 라벤더 반투명, radius 20px 20px 20px 4px, 패딩 16px, 최대 너비 75%, Nunito 15px, 흰색 텍스트. 말풍선 하단 왼쪽에 `"🔊 듣기"` 버튼 (골드 색상, 12px)
- **학생 말풍선**: 오른쪽 정렬, 배경 `rgba(255,255,255,0.9)` 흰색 반투명, radius 20px 4px 20px 20px, 패딩 16px, 최대 너비 75%, Nunito 15px, 어두운 라벤더 텍스트

### 하단 입력창 (고정)
- 화면 하단 고정, 배경 `rgba(255,255,255,0.08)` glassmorphism + blur 20px, 패딩 16px 20px
- 왼쪽: 마이크 아이콘 버튼 (동그란 버튼, 라벤더 배경, 44×44px)
- 중앙: 텍스트 입력창 (pill 형태, 흰색 10% opacity 배경, radius 999px, placeholder `"무니에게 설명해줘..."`)
- 오른쪽: 전송 버튼 (골드 색상 화살표 아이콘, 44×44px)

### 특수 효과
- 별빛 파티클: 3~5개의 ✦ 기호가 캐릭터 주변에 떠있음 (골드, 작게, 투명도 varies)
- 전체적인 느낌: Duolingo의 캐릭터 중심 + 달밤의 신비로운 분위기
