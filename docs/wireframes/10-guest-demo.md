# 10. 게스트 데모 (심사위원용)

## 목적
심사위원이 로그인 없이 서비스 핵심 기능을 양쪽(학생/선생님) 모두 체험

## ASCII 와이어프레임

```
┌─────────────────────────┐
│  🌙 무니에게 알려줘 데모  │
│                          │
│  어떤 역할로 체험할까요?  │
│                          │
│  ┌───────────────────┐   │
│  │ 🐇 학생으로 체험   │   │  ← 학생 데모
│  │ 무니에게 분수를    │   │
│  │ 설명해보세요       │   │
│  └───────────────────┘   │
│                          │
│  ┌───────────────────┐   │
│  │ 📚 선생님으로 체험 │   │  ← 선생님 데모
│  │ 학생 리포트를      │   │
│  │ 확인해보세요       │   │
│  └───────────────────┘   │
│                          │
│  * 데모 데이터로 체험     │
│    실제 데이터 저장 없음  │
└─────────────────────────┘
```

## 데모 플로우

```
학생 데모:
/demo/student → 대화 화면 (미리 설정된 "분수의 덧셈" 단원)
             → 실제 AI 대화 가능
             → 세션 종료 → 샘플 리포트

선생님 데모:
/demo/teacher → 대시보드 (샘플 학생 5명, 샘플 데이터)
             → 학생 클릭 → 상세 리포트
```

## Stitch 프롬프트

**앱 타입**: 역할 선택 온보딩 화면 (375px)
**무드**: 랜딩과 연결되는 달밤 배경

### 배경
랜딩 히어로와 동일한 우주 배경

### 카드 2개
- 학생 카드: 무니 작은 이미지 포함, 라벤더 배경
- 선생님 카드: 책 이모지, 더 어두운 라벤더
- 각 카드: radius 24px, glassmorphism, tap하면 해당 데모로 진입
