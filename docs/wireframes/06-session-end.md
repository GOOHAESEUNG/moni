# 06. 세션 종료 + 자기성찰

## ASCII 와이어프레임

```
┌─────────────────────────┐
│  ✦  ★  ✦               │  ← 별빛 (달밤 배경 유지)
│                          │
│  ┌───────────────────┐   │
│  │   [무니 happy]    │   │  ← 무니 impressed 표정
│  └───────────────────┘   │
│  "잘 가르쳐줬어요! 🌙"    │
│                          │
│  ┌───────────────────┐   │
│  │     이해도         │   │  ← 점수 카드
│  │      78점          │   │
│  │  ████████░░       │   │
│  │  "분수 개념은 잘   │   │
│  │   이해했어요!"     │   │
│  └───────────────────┘   │
│                          │
│  ────오늘의 빈틈────       │
│                          │
│  ┌───────────────────┐   │
│  │ 😮 통분 개념       │   │  ← 약점 태그
│  │ 막혔던 부분이에요  │   │
│  └───────────────────┘   │
│                          │
│  스스로 돌아보기          │  ← 자기성찰 (논문 기반)
│  ┌───────────────────┐   │
│  │ 오늘 가장 어려웠던 │   │
│  │ 부분은?            │   │
│  │ _________________ │   │  ← 1줄 입력
│  └───────────────────┘   │
│                          │
│  [리포트 보기] [홈으로]   │  ← 하단 버튼
└─────────────────────────┘
```

## Stitch 프롬프트

**앱 타입**: 모바일 결과 화면 (375px)
**무드**: 달밤 배경 유지 → 따뜻한 축하 분위기

### 배경
대화 화면과 동일한 우주 배경 유지. 상단에 골드 반짝임 효과 추가.

### 무니 캐릭터
- impressed 또는 happy 표정, 160×160px
- "잘 가르쳐줬어요! 🌙" 텍스트 (Nunito Bold 20px, 흰색)

### 점수 카드
- glassmorphism 카드 (white 15% opacity, blur)
- 점수 숫자: Nunito Black 48px, 골드 색상
- 프로그레스 바: 골드 그라디언트
- 코멘트 텍스트

### 약점 태그
- 섹션 "오늘의 빈틈"
- 빨간 대신 따뜻한 오렌지/골드 pill 태그 (디자인 원칙: 학생은 항상 이긴다)
- "다음에 더 잘할 수 있어요!" 격려 문구

### 자기성찰 입력
- 간단한 텍스트 입력창
- 입력 후 리포트에 포함됨을 안내

### 하단 버튼 2개
- "리포트 보기": 라벤더 border 버튼
- "홈으로": 라벤더 퍼플 filled 버튼
