# 07. 학생 리포트

## ASCII 와이어프레임

```
┌─────────────────────────┐
│  ← 분수의 덧셈 리포트    │
│  2026.04.08             │
│                          │
│  ┌───────────────────┐   │
│  │ 이해도 점수        │   │
│  │      78점          │   │
│  │  ██████████░░     │   │
│  └───────────────────┘   │
│                          │
│  ────잘한 부분────         │
│  ┌───────────────────┐   │
│  │ ✅ 분수의 기본 개념│   │
│  │ ✅ 분자/분모 이해  │   │
│  └───────────────────┘   │
│                          │
│  ────더 알아볼 부분────    │
│  ┌───────────────────┐   │
│  │ 💡 통분 개념       │   │  ← 빨간색 대신 파란 💡
│  │ "분모를 같게 만드는│   │
│  │  과정을 다시 연습해│   │
│  │  보면 좋아요"      │   │
│  └───────────────────┘   │
│                          │
│  ────대화 요약────         │
│  ┌───────────────────┐   │
│  │ 무니: "1/2+1/3은?" │   │  ← 핵심 대화 스니펫
│  │ 나: "2/5요!"       │   │
│  │ → 오개념 발견 지점 │   │
│  └───────────────────┘   │
│                          │
│  ────내가 한 말────        │
│  "통분이 어려웠어요"       │  ← 자기성찰
│                          │
│  [선생님께 공유] (선택)    │
└─────────────────────────┘
```

## Stitch 프롬프트

**앱 타입**: 모바일 학습 리포트 화면 (375px)
**무드**: 밝고 정돈된, 성취감을 주는 구조

### 배경
크림 화이트, 스크롤 가능한 페이지

### 점수 카드
- 상단 헤더 카드: 라벤더 그라디언트 배경
- 큰 점수 숫자 + 프로그레스 바

### 잘한 부분 / 더 알아볼 부분
- 두 섹션을 같은 스타일로 구성 (긍정 편향)
- 잘한 부분: ✅ 그린 체크 pill
- 더 알아볼 부분: 💡 골드 아이콘 (빨간색/경고 스타일 금지)
- 각 항목에 구체적인 설명 포함

### 대화 요약
- 채팅 스타일 미니 프리뷰 (실제 대화의 핵심 부분)
- 오개념이 발견된 순간 하이라이트

### 자기성찰
- 학생이 입력한 성찰 텍스트 강조 표시 (quote 스타일)
