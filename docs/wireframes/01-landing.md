## Stitch 프롬프트

**앱 타입**: 모바일 랜딩 페이지 (375px), 스크롤 가능
**무드**: 달밤 우주 배경 + 따뜻한 라벤더, 첫인상에서 "와" 나오는 화면

### 히어로 섹션
- 배경: 우주 딥 네이비 그라디언트 + 별빛
- 달 위의 무니 이미지 240px, float 애니메이션
- "무니에게 알려줘" 로고 텍스트 (Nunito Black 28px, 흰색)
- 헤드라인: "학생이 AI를 가르치며 / 진짜 실력이 쌓여요" (28px Bold, 흰색)
- CTA: "시작하기" (골드 배경), "체험해보기" (glassmorphism border)

### 기능 섹션 (라이트 배경으로 전환)
- 크림 화이트 배경
- 2×2 그리드 기능 카드, 이모지 + 타이틀 + 한줄 설명

### 연구 근거 섹션
- "Stanford 연구팀이 증명한 학습법"
- 효과 크기 수치 (크고 굵게)
