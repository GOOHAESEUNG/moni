# 08. 선생님 대시보드

## 사용자 & 목적
- **사용자**: 선생님
- **목적**: 반 전체 학생 이해도 한눈에 파악 + 관리
- **감정 목표**: 신뢰감, 효율적으로 다 보인다는 안도감

## Stitch 프롬프트

**앱 타입**: 모바일 교육 관리 대시보드 (375px)
**무드**: 신뢰감 있는 라벤더, 클린하고 스캔하기 쉬운 정보 구조

### 배경
크림 화이트 `#FAF9FF`, 전체적으로 밝고 깔끔함.

### 헤더
- "홍길동 선생님" Nunito Bold 22px + "나의 반 관리" muted 14px
- 오른쪽에 알림 아이콘

### 통계 그리드 (2×2)
- 4개 미니 카드: 숫자(크게) + 라벨(작게)
- 카드: 흰색 배경, radius 16px, 컬러 그림자
- 숫자: Nunito Bold 28px, 라벤더 퍼플
- 라벨: 12px, muted

### 활성 단원 카드
- 라벤더 배경 카드 (primary/10 opacity)
- 단원명 + 완료 학생 수 + 평균 점수
- 컬러 프로그레스 바 (라벤더→퍼플 그라디언트)

### 학생 목록
- 각 행: 아바타 + 이름 + 점수 + 상태 dot
- 상태 dot: 🟢 90+ / 🟡 70-89 / 🔴 70미만
- 점수 낮은 학생 행 배경 옅은 빨간 틴트
- 탭하면 학생 상세 리포트로 이동

### 하단 FAB
- 우하단: "+" 버튼 (라벤더 퍼플, 56px circle, 그림자)
