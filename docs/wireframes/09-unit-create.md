# 09. 단원 생성

## ASCII 와이어프레임

```
┌─────────────────────────┐
│  ← 새 단원 만들기        │  ← 헤더 + 뒤로가기
│                          │
│  ┌───────────────────┐   │
│  │ 단원 제목          │   │  ← 입력 필드
│  │ "분수의 덧셈"      │   │
│  └───────────────────┘   │
│                          │
│  ┌───────────────────┐   │
│  │ 오늘 배운 개념     │   │  ← 텍스트 에어리어 (핵심)
│  │ "분모가 다른 분수를│   │
│  │  더하려면 통분이   │   │
│  │  필요해. 1/2+1/3  │   │
│  │  = 3/6+2/6 = 5/6" │   │
│  │ (무니가 이걸 기반  │   │
│  │  으로 질문해요)    │   │
│  └───────────────────┘   │
│                          │
│  학년 힌트 (선택)         │
│  ○ 초등 저  ● 초등 고     │
│  ○ 중등    ○ 고등         │
│                          │
│  ┌───────────────────┐   │
│  │ 미리보기           │   │  ← 무니가 할 첫 질문 예시
│  │ 무니: "분수가 뭔지 │   │
│  │ 알려줄 수 있어? 🌙"│   │
│  └───────────────────┘   │
│                          │
│  [단원 저장하기]          │  ← CTA
└─────────────────────────┘
```

## Stitch 프롬프트

**앱 타입**: 모바일 폼 화면 (375px)
**무드**: 깔끔하고 집중적, 입력에 방해가 없는 구조

### 배경
크림 화이트, 미니멀

### 입력 필드
- 단원 제목: Nunito 16px, 라벤더 border, radius 14px, focus 시 라벤더 ring
- 개념 설명: 텍스트에어리어 5줄, 같은 스타일, placeholder 설명 포함
- 힌트 텍스트: "무니가 이 내용을 기반으로 질문을 만들어요"

### 학년 힌트 선택
- 가로 pill 형태 라디오 버튼 4개
- 선택된 것: 라벤더 배경 + 흰 텍스트
- 미선택: 연라벤더 border

### 미리보기 카드
- 옅은 라벤더 배경 카드
- 무니 작은 아이콘 + "무니가 이렇게 시작해요"
- 첫 질문 예시 텍스트

### CTA 버튼
- 전체 너비, 라벤더 퍼플, radius 999px, Nunito Bold 16px
